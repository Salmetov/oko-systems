Настоящее Лицензионное соглашение (далее — Соглашение) заключается между ИП Салметов М. (далее — Лицензиар) и Пользователем программы для ЭВМ «Oko Systems» (далее — ПО).

Основные термины

ПО — программа для ЭВМ «Oko Systems», исключительные права на которую принадлежат Лицензиару.

Демо-версия — версия ПО с ограниченным сроком использования, предназначенная для ознакомления с функциональными возможностями.

1. Предмет соглашения

1.1. Лицензиар предоставляет Пользователю право использования ПО (простая неисключительная лицензия) путём доступа к платформе через веб-интерфейс в соответствии с условиями настоящего Соглашения.

1.2. Настоящее Соглашение заключается до или непосредственно в момент начала использования ПО и действует на протяжении всего срока действия исключительного права Лицензиара на ПО.

1.3. Лицензиар предоставляет Пользователю право использования ПО на территории Республики Казахстан, Российской Федерации и иных стран СНГ.

2.2. ПО содержит коммерческую тайну и конфиденциальную информацию, принадлежащую Лицензиару.

2.3. Лицензиар гарантирует, что обладает всеми правами, необходимыми для предоставления Пользователю прав на использование ПО.

3. Условия использования и ограничения

3.1. Настоящее Соглашение предоставляет право доступа и использования ПО в рамках его функциональных возможностей для одного подключённого портала Bitrix24.

3.2. Запрещается копировать, модифицировать, распространять, продавать или сдавать в аренду любую часть ПО.

3.3. Запрещается осуществлять обратную разработку (reverse engineering) или попытки извлечения исходного кода ПО.

3.5. Запрещается любое использование ПО, противоречащее действующему законодательству.
